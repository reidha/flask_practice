## Environment

### Virtual environment

```
# Create a new environment
$ git clone 
$ python3 -m venv .venv

# Activate
$ source .venv/bin/activate

# Install libraries
$ pip install -r requirements.txt

# Deactivate
(.venv) $ deactivate
```

### Create requirements.txt

```
$ pip freeze > requirements.txt  
```
