## Environment

https://docs.python.org/3/tutorial/venv.html

```
# Create a new environment
$ git clone 
$ python3 -m venv .venv

# Activate
$ source .venv/bin/activate

# Install libraries
$ pip install -r requirements.txt

# Deactivate
(.venv) $ deactivate
```

```
$ pip freeze > requirements.txt  
```
